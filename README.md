# Peak Escape

A 2D climbing game made with Godot 4.

Climb terrain, avoid hazards, use ropes and checkpoints, and reach the summit.

## Requirements

- [Godot 4.x](https://godotengine.org/) (project uses GL Compatibility renderer)

## How to run

1. Install Godot 4.
2. Clone this repo and open the project folder in Godot.
3. Press **Play** (F5) or run the main scene.

## Project structure

- `scenes/` — Main menu and game scenes
- `scripts/` — Autoloads (GameManager, SaveManager, AudioManager), player, world (level generator, terrain, hazards, pickups), and UI
